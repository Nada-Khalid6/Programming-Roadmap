## ✨ المميزات

- 🎯 **مسارات تعلم متسلسلة** - تعلم البرمجة بطريقة منظمة وتدريجية
- 📺 **قوائم تشغيل يوتيوب مختارة** - أفضل الموارد التعليمية العربية
- ☑️ **نظام تتبع التقدم** - شيكليست لتسجيل إنجازاتك
- 🌐 **واجهة سهلة الاستخدام** - مصممة للمبتدئين
- 📊 **إحصائيات التقدم** - شاهد نسبة إكمالك للمسار
- 🇸🇦 **محتوى عربي بالكامل** - كل شيء باللغة العربية

## 🚀 البدء السريع

### المتطلبات
- Python 3.8 أو أحدث
- pip (مدير حزم Python)

### التثبيت

1. **استنسخ المستودع:**
```bash
git clone https://github.com/Nada-Khalid6/Programming-Roadmap.git
cd Programming-Roadmap
```

2. **ثبت الحزم المطلوبة:**
```bash
pip install -r requirements.txt
```

3. **شغّل التطبيق:**
```bash
streamlit run app.py
```

4. **افتح المتصف��:**
التطبيق سيفتح تلقائياً على `http://localhost:8501`

## 📚 المسارات المتاحة

مسار شامل للمبتدئين يغطي:
- مقدمة في البرمجة ومفاهيم أساسية
- تعلم لغة Python من الصفر
- الخوارزميات المنطقية وحل المشكلات

### 2. 🌐 تطوير مواقع الويب
مسار متكامل لبناء مواقع ويب:
- أساسيات HTML و CSS
- JavaScript للمبتدئين

2. **ابدأ الرحلة** - اتبع المراحل بالترتيب
3. **افتح الموارد** - اضغط على الأزرار لفتح قوائم ا��تشغيل

## 🛠️ التخصيص

يمكنك تعديل المسارات والموارد بسهولة:

### إضافة مسار جديد

عدّل `app.py` وأضف في القسم `tracks`:

```python
{
    "id": 3,
    "name": "اسم المسار",
    "description": "وصف المسار",
    "icon": "🎯"
}
```

ثم أضف المراحل في `roadmaps`:

```python
"3": [
    {
        "title": "عنوان المرحلة",
        "playlist_url": "رابط قائمة التشغيل",
        "description": "وصف المرحلة",
        "completed": False
    }
]
```
